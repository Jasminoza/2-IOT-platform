# IoT Microservice Platform
Это учебный проект, демонстрирующий архитектуру микросервисов для Internet of things (IoT). 

Содержит инфраструктуру (Kafka, PostgreSQL, и др.) и пример реализации процессов IoT.

## Структура репозитория

В корне проекта находятся:
1) docker-compose.yaml - файл для быстрого поднятия инфраструктуры проекта.
2) Makefile - файл для быстрого запуска отдельных микросервисов либо всей платформы.

В папке /architecture находятся основные диаграммы платформы.
В каждой отдельной папке находится исходный код отдельного микросервиса, который включает в себя Dockerfile и Makefile

## Подготовка к запуску
Скачайте и установите необходимые инструменты:
1) sdkman
2) java 25
3) docker

## Настройка
Скопируйте .env.example в .env и укажите необходимые переменные (или используйте значения по умолчанию)


## Запуск
Запустите платформу
```shell
  Makefile start platform
```

## Как проверить работу
После запуска можно убедиться в работоспособности платформы через

Grafana (admin/admin) - http://localhost:3000

Keycloak (admin/admin) - http://localhost:8080
